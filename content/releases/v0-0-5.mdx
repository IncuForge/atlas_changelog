## Full Pi slash command support

Compass now integrates all official Pi slash commands directly, so anything you could run through Pi itself is available without leaving Compass.

## Session trees, forking, and queues

Compass gains native support for Pi's follow-up queues, compaction state, session trees, and forking, making it easier to branch conversations and manage longer-running work.

## Richer profiles

Profiles have been expanded to cover model selection, web access, MCP configuration, and skill controls, giving each profile a more complete picture of how it should behave.

## Reusable model sets

Reusable Pi model sets let you define scoped groups of models once and apply them across workflows, instead of reconfiguring model choices each time.

## Better provider and model discovery

Provider discovery, custom provider configuration, and model selection have all been improved, making it easier to find and choose the right provider and model for the job.

## Native extension notifications

Compass now natively handles Pi extension notifications and interactive choices, so extensions can surface prompts and updates directly in the interface.

## Refreshed interfaces

Onboarding, Herakles themes, and Compass's settings interfaces have all been refreshed for a cleaner, more consistent experience.

## Steadier sessions and runtime

Session restoration, context reporting, and overall runtime reliability have been improved, reducing rough edges when resuming or monitoring long sessions.

## Simpler packaging

Release packaging has been simplified, and builds now include SHA-256 artifact manifests for easier verification.
