## A more dependable workspace

Atlas continues to mature as the local desktop companion for Pi. This release focuses on making the experience easier to understand, easier to recover from when something goes wrong, and more transparent about how Atlas and Pi work together.

## Better visibility, safer sessions

Compass now reports Pi installation details, compatibility, and provider diagnostics without exposing credentials or conversation content. Shared Pi sessions can reopen in their original working directories, and conversation deletion now clearly distinguishes removing Atlas metadata from permanently deleting Pi session data.

## A more polished Atlas

The interface has been refined with updated branding, a clearer welcome experience, improved About page behavior, and a more subtle Agent View watermark. Release automation now targets macOS Apple Silicon, Windows x64, and Linux x64.
